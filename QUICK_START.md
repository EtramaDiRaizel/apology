## 🚀 Installation & Running

### Step 1: Install Node Packages
```bash
cd C:\Users\ikram\Downloads
npm install
```
*This installs React, Framer Motion, Tailwind CSS, and development tools (~500MB)*

### Step 2: Start Development Server
```bash
npm run dev
```
**Expected output:**
```
  VITE v4.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  press h to show help
```
Browser opens automatically to the slideshow.

### Step 3: Navigate the Slides
- **Keyboard**: Arrow keys or Space
- **Scroll**: Mouse wheel
- **Buttons**: Left/right arrows on screen

## 🔧 Customization

### Change Colors
Edit `tailwind.config.js`:
```js
colors: {
  'charcoal': '#111111',      // Change here
  'off-white': '#F5F5F7',     // Or here
  'subtle-accent': '#D4AF37', // Or here
}
```

### Edit Slide Content
- **Slide 1**: Edit `src/slides/Slide1Title.jsx` — title text
- **Slide 5 Apology**: Edit paragraphs in `src/slides/Slide5Apology.jsx`
- Other slides consume dynamic data from `wrapped_data.json`

### Adjust Animation Timing
In any slide component, modify `transition` properties:
```jsx
transition={{ duration: 0.6, ease: 'easeOut' }} // Change 0.6 to speed up/slow down
```

**Issue**: Module not found errors
```bash

```

**Issue**: Styles not loading
```bash

```
